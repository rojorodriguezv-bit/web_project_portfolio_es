# Portafolio Web de Rojo Rodríguez

Este proyecto es un portafolio web diseñado para una desarrolladora frontend. Está pensado para mostrar proyectos, habilidades y datos de contacto con un diseño limpio, intuitivo y adaptado a móviles.

## Tecnologías
- HTML
- CSS
- Diseño responsivo
- GitHub Pages

## Características
- Diseño adaptable a móviles, tablets y escritorio
- Secciones de bienvenida, proyectos, habilidades y contacto
- Uso de fuentes personalizadas y colores agradables
- Navegación sencilla y clara
